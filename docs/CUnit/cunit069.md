# 0x0E1: wireframeRandomizer

| Name | wireframeRandomizer |
| ----| ------------ |
| Offset | 0x0E1 |
| Type | u8 |
| Mask | 0x0000FF00 |
| Description | 유닛이 테란/프로토스 와이어프레임을 가진 경우, 부상 시 어떤 부위에 노랑/빨강이 칠해질지를 결정하는 요소. 홀수/짝수에 따라 사용할 색들이 결정되며, 자세한 내용은 [링크 글](https://cafe.naver.com/edac/127264) 참고.  |<br>
