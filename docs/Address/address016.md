# Active Storm ID

| `Address` | `Player ID` | `Version` | `Size` | `Length` | `SCR` |
| ---------- | ----------- | --------- | ------ | -------- | ---- |
| 0x512680 | -122681 | 1.16.1 | 4 | 1 | Unsupported |

# Description

This is your active storm ID, which is different from your player ID. The official name for this is g_ActiveStormID.
