# CUnit - Previous HP

| `Address` | `Player ID` | `Version` | `Size` | `Length` | `SCR` |
| ---------- | ----------- | --------- | ------ | -------- | ---- |
| 0x59CD56 | 19068 | 1.16.1 | 2 | 1 | Supported |

# Description

The HP of the unit before it changed (example Drone->Hatchery, the Drone's HP will be stored here)
